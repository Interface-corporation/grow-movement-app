# Seed Fund partner logos

Drop real partner logo files in this folder using the exact filenames below.
The Seed Fund page renders an automatic "Drop logo here" placeholder for any
that aren't present yet, so the page never breaks.

Required files (PNG or SVG, transparent background recommended, ~400×160 px):

- linklaters.png
- bcg.png
- lbs.png
- lse.png
- cbs.png
- cems.png
- ncif.png

To add more partners, edit the `partners` array in `src/pages/SeedFund.tsx`.
